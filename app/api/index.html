Tony Tech Insights API

This is the API server for Tony Tech Insights. The following endpoints are available:

GET /api/

Health check endpoint that returns the status of the API.

{
    "status": "ok",
    "message": "Simple API is running",
    "environment": "production",
    "python_version": "3.9.x"
}

GET /api/test

Test endpoint to verify the API is working correctly.

{
    "status": "ok",
    "message": "Test endpoint is working correctly"
}

POST /api/chat

Chat endpoint that accepts a prompt and returns a response.

Request body:

{
    "prompt": "Your message here",
    "session_id": "optional-session-id"
}

Response:

{
    "response": "AI response to your message",
    "session_id": "session-id",
    "tool_calls": [],
    "request_id": "request-id"
}

For more information, please contact the administrator.
